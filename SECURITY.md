# Security Policy

## Supported Versions

CTEM Leader Lab is currently pre-1.0. Security fixes are accepted for the latest release line.

| Version | Supported |
| --- | --- |
| v0.1.x | Yes |

## Reporting a Vulnerability

Please do not open a public issue for a vulnerability that could affect users of this project.

Use a private GitHub security advisory if the repository is hosted on GitHub, or contact the maintainers through the project owner account. Include:

- A clear description of the issue.
- Reproduction steps.
- Affected file, route, or dependency.
- Potential impact.
- Suggested fix, if known.

## Safety Statement

This project is a fictional CTEM learning workbench. It must not be used to test, scan, exploit, or assess systems without explicit authorization. Contributions that introduce offensive automation or unsafe instructions will be rejected.
